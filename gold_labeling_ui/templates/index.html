Labeling Rubric — criteria for is_correct, type, Bloom, domain

Skills — is_correct

yes: Real skill from the posting. no: Garbage, fragment, not a skill, or hallucinated.

Skills — type_label & bloom_label

Type: Hard / Soft / Both. Bloom: Remember, Understand, Apply, Analyze, Evaluate, Create (hard skills); leave blank if unsure or soft.

Knowledge — is_correct

yes: Real concept from posting. no: Garbage, fragment, or hallucinated.

Domain — true_domain_id

Use domain IDs from future_domains.csv. none = no fit. unclear = undecidable.
